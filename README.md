# Sistema de Cadastro e Análise de Produtos

Este projeto consiste em um sistema desenvolvido em Python para o registro de produtos e análise de produtos de uma loja. O objetivo é aplicar conceitos fundamentais de programação, como estruturas de repetição, funções, tratamento de erros e manipulação de dados com dicionários e listas.

## 📋 Funcionalidades

O sistema permite realizar as seguintes operações:

1.  **Cadastro de Produtos:** Registro de produtos, coletando nome e preço.
2.  **Cálculo de Estatísticas:**
    *   Soma total dos preços.
    *   Cálculo da média de preços.
3.  **Validação de Entradas:** Uso de tratamento de exceções (`try-except`) e laços `while` para garantir que os preços informados sejam numéricos.
4.  **Relatório Final:** Exibição detalhada de todos os produtos cadastrados e os resultados das análises.

## 🛠️ Tecnologias Utilizadas

*   **Python 3.x**
*   **Dicionários e Listas** para organização de dados.


## 📝 Requisitos da Atividade

O projeto foi desenvolvido seguindo os critérios de:
*   **Organização e Modularização:** Código limpo e funções bem definidas.
*   **Legibilidade:** Nomes de variáveis intuitivos e comentários relevantes.
*   **Tratamento de Erros:** Robustez na entrada de dados numéricos.
